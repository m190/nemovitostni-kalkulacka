# Nemovitostní kalkulačka

Kalkulačka zhodnocení investice do nemovitosti (IRR, cashflow, odpisy a úroky pro daňové účely,
srovnání s alternativní investicí do akcií/ETF). Jeden samostatný HTML soubor, žádné závislosti.

## Použití lokálně
Otevřete `index.html` v prohlížeči.

## Nasazení na GitHub Pages
1. Settings → Pages → Source: nastavit na branch `main`, folder `/ (root)`.
2. Po uložení bude web dostupný na `https://<uzivatel>.github.io/<nazev-repozitare>/`.
